💡 ULTIMATE GLOSSARY FOR AUDIENCE QUESTIONS
================================================================================

1. JWT (JSON Web Token) - Digital Security Passport.
2. SHA-256 (Secure Hash Algorithm) - Digital Fingerprint Seal.
3. API (Application Programming Interface) - Software Translator/Bridge.
4. ERP (Enterprise Resource Planning) - Main software like SAP/Oracle.
5. PINT AE (Peppol International UAE) - The official UAE Invoice Standard.
6. VAT (Value Added Tax) - The 5% UAE consumption tax.
7. TRN (Tax Registration Number) - The 15-digit Tax ID.
8. SQL (Structured Query Language) - The system used to save audit data.
9. 5-Corner Model - The UAE framework for sending/clearing invoices via ASPs.
